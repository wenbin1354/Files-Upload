# Files-Upload
React + Node.js + Express + Multer + Mongodb to handle file uploads

for client
```bash
# api-backend terminal 1
cd client
npm install
npm start
```

for server run
```bash
# react-client terminal 2
cd api
npm install
node src/server.js
```

server will be hosted at http://localhost:8080
client will be hosted at http://localhost:3000
